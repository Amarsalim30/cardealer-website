# Feature Roadmap

## Phase 1
- Public website pages
- Inventory listing and vehicle detail flow
- SEO metadata, structured data, sitemap, and robots
- Quote, contact, financing, test drive, and trade-in forms
- Minimal admin login and vehicle management
- Leads inbox

## Phase 2
- Better inventory sorting and saved filter experiences
- Review management in admin
- More robust location management
- Marketplace feed export
- Richer analytics and attribution reporting

## Phase 3
- Staff roles and permissions
- Lead assignment and status tracking
- Import request workflow with calculators
- Reservation deposit handling
- Customer accounts or saved vehicles
