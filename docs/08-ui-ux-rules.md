# UI / UX Rules

## Design Direction
- Clean, premium, and credible
- Warm-neutral backgrounds with dark, readable typography
- One consistent primary CTA style across the site
- Motion used sparingly and only to support hierarchy

## Spacing and Layout
- Use generous spacing on desktop and compact stacks on mobile
- Keep key content within readable max widths
- Use section headers with a short eyebrow, strong title, and one supporting sentence

## CTA Rules
- Primary CTA appears above the fold on vehicle detail pages
- Phone and WhatsApp remain highly visible across the site
- On mobile, keep the sticky CTA bar limited to the highest-value actions
- Use action-first labels, not vague text like "Learn more"

## Card Rules
- Vehicle cards must always show image, title, price, specs, location, and CTA
- Featured cards can have stronger visual treatment but must stay readable
- Avoid dense spec walls on cards

## Form Rules
- Keep lead forms short and low-friction
- Default fields: name, phone, email, message
- Only ask for extra fields when required by the form purpose
- Show clear success and error feedback

## Image Rules
- Vehicle card images should use a consistent landscape ratio
- Vehicle detail galleries must handle missing or partial image sets gracefully
- Use helpful alt text and avoid decorative-only galleries

## Mobile Rules
- Navigation must be usable one-handed
- Sticky lead actions should not block core content
- Filters and forms must remain usable without precision taps
